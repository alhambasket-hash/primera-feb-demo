Para presentar a clubes

Que se pediria para pasar de demo a web real

Esta demo usa identidades y retratos generados. La version final necesitaria autorizacion de cada club para publicar escudo, fotos de plantilla, pabellon y material de prensa.

Material visual Escudo, fotos oficiales de jugadores, staff y pabellon.
Uso permitido Web informativa no oficial, con atribucion y enlaces al club.
Actualizacion Plantilla y fotos revisables por el departamento de comunicacion.
Control Retirada o sustitucion de imagenes si el club lo solicita.
Demo no oficial creada para solicitar autorizaciones de clubes. Datos, logos y retratos son ficticios.
